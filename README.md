# ⚡ AI Vision System (Cognitive Vision Platform)

🚀 An advanced AI-powered real-time computer vision system designed for intelligent surveillance, analytics, and automated insights.

---

## 🌟 Overview

This project is a **modular AI vision platform** that performs real-time object detection, tracking, behavior analysis, and alert generation using YOLOv8.

It combines **computer vision + analytics + UI dashboard + voice feedback** into a single powerful system.

---

## 🔥 Features

### 🎯 Core Capabilities
- Real-time object detection using YOLOv8
- Multi-object tracking with persistent IDs
- Multi-camera support
- Zone-based intrusion detection

---

### 🧠 AI Intelligence
- Scene understanding (auto-generated explanations)
- Behavior detection:
  - Crowd detection
  - Group activity detection
  - Loitering detection
- Smart alerts (zone entry, abnormal activity)

---

### 📊 Analytics Dashboard
- Object count tracking
- FPS monitoring (performance)
- Real-time trend graphs
- Detection history logging
- Visual analytics (charts & summaries)

---

### 🔊 Voice Assistant
- Real-time voice alerts (e.g., crowd detected)
- Toggle ON/OFF control
- Multi-voice support (system voices)

---

### 📸 Utilities
- Capture frames (manual + auto on alerts)
- Download processed images
- Export logs (CSV format)

---
